# Baby Companion Web

一个给家人用的宝宝陪伴 Web MVP，先把三件高频需求收进一个轻量原型：

1. 哄睡音乐
2. 照片 / 视频记录整理
3. 宝宝教育清单 / 代办

## 当前进展

- ✅ 落地首版静态 Web 原型
- ✅ 提供三个核心模块的页面结构与交互骨架
- ✅ 内置示例数据，打开即可演示

## 目录

- `index.html`：页面骨架
- `styles.css`：界面样式
- `app.js`：前端交互与示例数据

## 运行

直接双击 `index.html`，或在目录下启动本地静态服务：

```bash
cd baby-companion-web
python3 -m http.server 4173
```

然后打开 <http://localhost:4173>。

## 模块

### 1) Sleep
- 白噪音 / 钢琴 / 雨声卡片
- 时长预设
- 最近一次哄睡记录

### 2) Memories
- 今日照片 / 视频记录卡片
- 标签筛选
- 按时间线浏览

### 3) Care
- 教育清单 / 疫苗 / 用品 / 陪伴任务
- 今日待办与优先级展示

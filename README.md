# Baby Companion Web

一个给家人用的宝宝陪伴 Web MVP，先把三件高频需求收进一个轻量、可直接演示和试用的本地面板：

1. 哄睡音乐
2. 照片 / 视频记录整理
3. 宝宝教育清单 / 代办

## 当前进展

- ✅ 移动端优先的单页仪表盘
- ✅ 三个核心模块都有可操作流程，不再只是静态卡片
- ✅ 使用 `localStorage` 保存记录、待办和哄睡历史
- ✅ 内置示例数据，打开即可演示
- ✅ 新增移动端粘性导航，单手切换哄睡 / 记录 / 教育清单
- ✅ 哄睡模块支持按场景切换推荐曲目与时长（夜哄 / 午睡 / 安抚哭闹）
- ✅ 成长记录新增媒体整理视图，快速切到照片 / 视频 / 重点内容
- ✅ 教育方向模板支持一键把语言启蒙 / 感官探索 / 大运动 / 精细动作加入清单

## 目录

- `index.html`：页面骨架
- `styles.css`：界面样式
- `app.js`：前端交互与示例数据

## 运行

直接双击 `index.html`，或在目录下启动本地静态服务：

```bash
cd baby-companion-web
python3 -m http.server 4173
```

然后打开 <http://localhost:4173>。

## 模块

### 1) Sleep
- 白噪音 / 钢琴 / 雨声 / 海浪等哄睡曲目轮换
- 时长预设选择
- 开始 / 完成一轮哄睡，并生成最近记录
- 最近哄睡记录列表

### 2) Memories
- 手动新增照片 / 视频 / 照片组 / 成长笔记
- 标签筛选
- 标记重点记录
- 按时间线浏览

### 3) Care
- 教育清单 / 疫苗 / 用品 / 陪伴任务
- 新增待办、负责人、截止时间、优先级
- 标记完成并切换进行中 / 已完成视图
